== Description ==
The Simple Country Blocker plugin allows you to restrict access to your WordPress website based on selected countries.

* [Need Help?](https://wordpress.org/support/plugin/simple-country-blocker)
* [About Author](https://github.com/md-arif-islam)

= Usage =

* After installation, navigate to "Settings > Simple Country Blocker" to configure the countries you want to block.

==  ⚡ Features ==
  * Restrict access to your website based on selected countries
  * Choose which countries to block
  * Easy-to-use settings page to manage the blocked countries
  * Uses the ipinfo.io API to get the user's country based on their IP address

== Installation ==

= Minimum Requirements =

* WordPress 5.2 or greater
* PHP version 7.2 or greater
* MySQL version 5.6 or greater

= AUTOMATIC INSTALLATION =

1. Download the plugin ZIP file.
2. In your WordPress admin dashboard, navigate to "Plugins > Add New".
3. Click on the "Upload Plugin" button and select the ZIP file you downloaded.
4. Click "Install Now" and then activate the plugin.

= MANUAL INSTALLATION =

The manual installation method involves downloading our Directory plugin and uploading it to your webserver via your favorite FTP application. 

1. Upload `simple-country-blocker` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

= UPDATING =

Automatic updates should work seamlessly. We always suggest you backup your website before any automated update to avoid unforeseen problems.
